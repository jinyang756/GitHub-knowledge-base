3. 更新根目录下的`_sidebar.md`文件，添加新文档的导航链接

## 内容规范

为了确保话术生成工具能够高效提取专业内容，请遵循以下规范：

- **标题层级**：使用清晰的标题层级（#、##、###等）组织内容
- **重点标记**：对于策略要点、关键指标等重要内容，请使用**加粗**或`代码块`进行标记
- **结构化表达**：尽量使用列表、表格等结构化形式呈现策略步骤和条件
- **回测数据**：提供策略回测结果时请注明时间范围、参数设置和绩效指标
