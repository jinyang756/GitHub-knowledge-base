## 内容规范

为了确保话术生成工具能够高效提取专业内容，请遵循以下规范：

- **标题层级**：使用清晰的标题层级（#、##、###等）组织内容
- **重点标记**：对于代码示例、技术参数、配置说明等重要内容，请使用**加粗**或`代码块`进行标记
- **结构化表达**：尽量使用列表、表格等结构化形式呈现技术信息
- **代码规范**：代码示例应遵循行业标准规范，添加必要的注释说明
- **工具介绍**：介绍工具时请注明版本、功能特点和使用场景
